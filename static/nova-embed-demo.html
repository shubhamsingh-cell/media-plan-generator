Nova AI Embed Widget

A lightweight, zero-dependency JavaScript widget that ATS vendors can embed to offer Nova-powered media planning directly in their platforms.

Quick Start

Add a single script tag to your page. The widget will appear as a floating button in the bottom-right corner. 

<!-- Minimal integration: one line of code -->
<script src="https://media-plan-generator.onrender.com/embed/nova.js"></script>

Configuration

Customize the widget using HTML data attributes on the script tag. 

<script src="https://media-plan-generator.onrender.com/embed/nova.js"
        data-nova-key="your-api-key"
        data-nova-theme="dark"
        data-nova-position="bottom-right"></script>
Attribute Values Default Description
data-nova-key string (none) API key for authenticated access
data-nova-theme dark / light dark Widget color theme
data-nova-position bottom-right / bottom-left bottom-right Floating button position
data-nova-base-url URL string (auto-detected) Override the Nova API endpoint

postMessage API

Control the widget programmatically from your host page.

// Open the widget
window.postMessage({ type: 'nova-embed-open' }, '*');

// Close the widget
window.postMessage({ type: 'nova-embed-close' }, '*');

// Pre-fill the form
window.postMessage({
  type: 'nova-embed-prefill',
  data: {
    jobTitle: 'Senior Data Engineer',
    budget: '8000',
    location: 'New York, NY'
  }
}, '*');

// Listen for events from the widget
window.addEventListener('message', function(ev) {
  if (ev.data.type === 'nova-embed-ready') { /* widget loaded */ }
  if (ev.data.type === 'nova-embed-plan-generated') { /* plan ready */ }
  if (ev.data.type === 'nova-embed-error') { /* error occurred */ }
});

ATS Integration Example

Here is how an ATS platform would integrate Nova to auto-populate the widget when a recruiter views a job requisition: 

// In your ATS job detail page
function onJobLoaded(job) {
  // Pre-fill Nova with job data from your ATS
  window.postMessage({
    type: 'nova-embed-prefill',
    data: {
      jobTitle: job.title,
      budget: job.recruitingBudget,
      location: job.location
    }
  }, '*');
}

// Listen for generated plans to save back to your ATS
window.addEventListener('message', function(ev) {
  if (ev.data.type === 'nova-embed-plan-generated') {
    // Save the media plan to your ATS database
    saveMediaPlanToATS(ev.data.data);
  }
});
