## Executive Summary

Nova AI Suite is a five-product recruitment intelligence platform built to answer one question: where should you spend your recruitment media budget, and why?

The suite combines 57 AI-powered tools, 22+ live government and commercial APIs, and a multi-provider LLM routing layer to deliver salary intelligence, labor market analytics, competitive positioning, media planning, and campaign optimization -- all from a single platform with shared data infrastructure.

**Products at a glance:**

| # | Product | Core Function | Status |
|---|---------|--------------|--------|
| 1 | Media Plan Generator | Budget-to-plan automation | Live (Render) |
| 2 | Nova AI Chatbot | Conversational recruitment intelligence | Live (Render) |
| 3 | GeoViz 3D | 3D geospatial labor market visualization | Live (Vercel) |
| 4 | CG Automation | Craigslist posting optimizer | Live (Render) |
| 5 | Platform & Intelligence Tools | Campaign, compliance, vendor, and market tools | Live (Render) |

**Infrastructure:** Supabase (22 tables), Qdrant (685 vectors), LLM Router (23 providers), 3-layer cache (in-memory + Supabase + Upstash Redis), Google Auth with Joveo SSO, PostHog analytics, Sentry error tracking.

## Product 1: Media Plan Generator

### What It Does

Takes three inputs -- role, location, and budget -- and produces a complete recruitment media plan with channel allocation, competitive analysis, risk assessment, and activation calendar. Output is generated in 60-90 seconds.

### Gold Standard Quality (7/7 Dimensions)

Every generated plan scores against seven quality dimensions:

| Dimension | What It Covers |
|-----------|---------------|
| City-level data | Salary, demand, and supply metrics specific to the target metro area |
| Security clearance | Clearance requirements when applicable to the role |
| Competitor analysis | 15+ metros across 7 industries with counter-strategies |
| Role difficulty | Difficulty scoring across 45+ role categories |
| Channel recommendations | Publisher selection with rationale explaining why each channel fits |
| Budget tiers | Allocation across tiers with trade-off analysis |
| Activation calendar | Week-by-week deployment schedule with milestone events |

### Output Formats

- **Excel workbook**: 6 sheets covering executive narrative, channel breakdown, budget allocation, competitive landscape, risk analysis, and activation timeline
- **PowerPoint deck**: 10+ slides including market thesis, risk analysis (2x2 severity cards), competitor positioning cards, and channel rationale
- **PDF report**: Branded professional document via reportlab
- **Shareable link**: Dashboard view with Open Graph meta tags for previews in Slack and LinkedIn

### Key Capabilities

- **C-suite narrative quality**: Every plan includes a synthesized market narrative generated by Claude Haiku, written for executive audiences -- not raw data dumps
- **Competitor counter-strategies**: Plans identify what competitors are doing in the same labor market and recommend differentiated approaches
- **Industry inference**: 50+ role-keyword mappings automatically classify roles into the correct industry vertical, eliminating manual categorization
- **Async pipeline**: 30-second ThreadPoolExecutor timeouts with deadline checks between stages prevent runaway generation
- **Synthesis engine**: Claude Haiku generates all narrative sections, ensuring consistent voice and analytical depth

## Product 4: CG Automation

### What It Does

Optimizes Craigslist ad campaign posting by analyzing historical performance data and producing per-location recommendations for title, category, day-of-week, and posting frequency. Replaces manual guesswork with data-driven daily action plans.

### Data Processing

| Metric | Value |
|--------|-------|
| Rows processed | 98,665 |
| Processing time | 8 seconds |
| Unique posts analyzed | 5,707 |
| Locations covered | 397 |
| Average post run length | 18.3 days |

### Core Business Logic

- **Sell CPA**: $1.20 per apply (fixed)
- **Gross Revenue** = Applies x $1.20
- **Net Revenue** = GR - Media Cost (cost deducted once on Day 1 only)
- **Location-specific multipliers**: Every location has its own lifetime multiplier. Global averages are never used. Fallback order: Location Average, then Category Average, then Global Average (3.60x).

### Repost Decision Engine

| Condition | Action |
|-----------|--------|
| Profit % < 0% | REPOST (fresh start) |
| Impression drop >= 90% AND profit >= 10% | REPOST (refresh) |
| Profit >= 10% AND 80% < drop < 90% | REPOST (decay) |
| Profit >= 10% AND drop <= 80% | Keep running |
| Profit < 10% or lifetime NR <= $0 | Skip |

### Priority Tiers

| Tier | Criteria | Action |
|------|----------|--------|
| Tier 1 | Day-1 Net Revenue > $0 | Post first |
| Tier 2 | Profit >= 100% | High priority |
| Tier 3 | Profit >= 50% | Medium priority |
| Tier 4 | Profit >= 10% | Standard priority |

### Per-Location Intelligence

All recommendations are location-specific, never global:

- Best title per location (by average Total NR)
- Best category per location (by average Total NR)
- Best day-of-week per location (by average Total NR)
- Best title x category combination per location
- Frequency optimization (maximizes absolute weekly NR, not profit percentage)

### v3.0 Interface (4 Consolidated Views)

| View | Purpose |
|------|---------|
| Dashboard | Portfolio-level KPIs, trend charts, priority distribution |
| Action Plan | Today's ranked posting recommendations with AI insights |
| Intelligence | Per-location drill-downs, title/category/day analysis |
| Reports | 9-sheet styled Excel export, historical performance |

### Technical Architecture

- **Backend**: FastAPI, Python 3.11, pandas, openpyxl, numpy
- **Frontend**: React + Tailwind CSS, dark theme
- **AI layer**: Claude Opus for per-recommendation insights + Nova data enrichment for salary, demand, and CPA context
- **API endpoints**: 13 total, including upload, analysis, insights, download, schedules, and webhooks
- **Supabase tables**: 5 (cg_jobs, cg_action_plans, cg_schedules, cg_uploads, cg_user_sessions)
- **Security**: CORS restricted to explicit origins, 50 MB upload limit, SSRF validation on webhooks, schedule reload on startup

## Key Differentiators

1. **Data density**: 22+ live APIs and 25+ knowledge base files produce answers grounded in current government and commercial data, not static training data.

2. **Parallel tool execution**: Nova runs up to 5 tools concurrently per query, reducing response times and increasing answer completeness.

3. **Quality enforcement**: Mandatory tool calls, minimum-3 tool requirement for complex queries, reject-and-retry logic, and post-processing with citation injection ensure consistently high output quality.

4. **Global coverage**: CareerJet (60+ countries), Eurostat (EU), UK ONS, StatCan (Canada), and GeoNames provide international labor market intelligence beyond US-only data.

5. **Per-location intelligence**: CG Automation and Media Plan Generator both operate at the individual location level. Recommendations are never based on global averages.

6. **Shared infrastructure**: All five products share a single Supabase instance, LLM router, cache layer, and authentication system. This eliminates data drift between products and reduces operational overhead.
