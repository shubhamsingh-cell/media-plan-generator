**Legend**:
- **YES** = Direct import and usage

```
+============================================================================+
|                           USER INPUT                                        |
|  (industry, roles, locations, budget, client_name, competitors)            |
+============================================================================+
                                    |
                                    v

|   standardizer.normalize_industry(industry)                                |

|   collar_intelligence.classify_collar(role, industry)                      |
|   _validate_locations(locations)                                           |
|                                                                            |
|   Output: Standardized inputs + collar classification                      |
+------------------------------------------------------------+---------------+
                                    |
                                    v
+---[data_orchestrator.py]--- ENRICHMENT LAYER (17 Functions) ---------------+
|                                                                            |
|   SALARY         enrich_salary ---------> BLS OES, Adzuna, H-1B           |
|   LOCATION       enrich_location -------> Census, GeoNames, Teleport      |
|   MARKET         enrich_market_demand --> JOLTS, FRED, O*NET, ILO         |
|   COMPANY        enrich_competitive ----> Wikipedia, SEC EDGAR            |
|   BRAND          enrich_employer_brand -> Clearbit, Logo.dev              |
|   AD BENCHMARKS  enrich_ad_benchmarks --> Google, Meta, Bing, TikTok, LI  |
|   COLLAR         enrich_collar_intel ---> collar_intelligence.py          |
|   TRENDS         enrich_hiring_trends --> trend_engine.py + FRED          |
|   BUDGET         enrich_budget ---------> budget_engine.py                |
|   + 8 more specialized functions (regulations, seasonal, campus, etc.)     |
|                                                                            |
|   All results: cached with TTL | request_id traced | error-tolerant       |
+------------------------------------------------------------+---------------+
                                    |
                                    v
+---[data_synthesizer.py]--- CROSS-SOURCE SYNTHESIS -------------------------+
|                                                                            |
|   - Merges all 17 enrichment outputs into unified structure                |

|   Output: Synthesized data object with confidence metadata                 |

                                    |

|     - Channel quality scoring (quality-of-hire, retention, time-to-hire)   |

|     - What-if simulation engine                                            |
|                                                                            |
|   Output: Allocation table + projections + quality scores                  |
+------------------------------------------------------------+---------------+
                                    |
                     +--------------+--------------+
                     |              |              |
                     v              v              v
+--- OUTPUT: 4 PRODUCTS --------------------------------------------------------+
```
